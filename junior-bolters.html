Project overview

About the Website: Junior Bolter believe that the best way to learn is when your child is having fun. Superbolter will create fun tasks for kids to complete. Each task has been thought through to improve cognitive thinking and creative imagination.

Here's the Website for Junior Bolter site which inform the users about all Competition that are currently going on and let them participate in it.  The Website needed the user to be able to submit via Website or WhatsApp chat or Mail to Showcase the designs. Brief was to plan content strategy & make it interactive and minimal. I did just that!

Website Walkthrough

The problem

Junior Bolters by SuperBolter needed a Website which will post Interior Design Challenges for Kids and they could participate in it. We needed to get the information right in the landing page that needed to explain what website is about and the benefit of platform. as well as how they can join the competition.

The website should as accept the Submission from User so that it can be displayed. Also the competitions should be marked as Live and Ended.

steps

I focused on Content Strategy first. Content is king and that is one of the important thing in design. So I first went through the various effective and award-winning site to understand how they are handling this use case.

Getting Inspiration

After considering and preparing the content for site along with the Founder, I started gathering inspiration from best sites that exist for this use case.

Wireframing & Design Process

After going through various sites layout, we finally decided how the site will look. Then I started to making wireframes to get rough idea of how the website will be laid out. Then I discussed it with the Client about it.

I moved on to Figma for visual design of the site. I used the best practices to keep the project organised.

Finally after several changes from various stakeholders on design part, I went to Webflow for implementation of the site. Here the task was to take the site one step ahead by adding motion and animation.

Development in Webflow

I used best practices to design the site on Webflow and the stakeholders were quite happy with the result at the first glance. Then we did some changes in fonts and other elements to make it look good on every device.
