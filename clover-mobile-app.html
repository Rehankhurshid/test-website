Project overview

Clover Mobile App UI Kit  helps customers to self-book visits to beauty salons, barbers, hairdressers, massage therapists, nail technicians, and many other appointment-based services in a simple and intuitive way, anytime they wish.

Customers can make a reservation by themselves and, if necessary, cancel the visit — regardless of the salon’s actual working hours, 24h a day.

It's available in Light as well as Dark Mode

This is UI Kit is available for sale under the Profile Name of SpearStudio which is an alias name under which I sell my work currently.
